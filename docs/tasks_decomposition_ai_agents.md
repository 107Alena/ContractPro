|  № | Агент                                        | Назначение                                                                                                                                                            |
| -: | -------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  1 | **Contract Type Classifier Agent**           | Определяет тип договора: услуги, поставка, подряд, аренда, NDA и др.; рассчитывает уровень уверенности классификации.                                                 |
|  2 | **Key Parameters Extractor Agent**           | Извлекает ключевые параметры договора: стороны, предмет, цену/порядок расчётов, сроки, ответственность, порядок приёмки, расторжение, применимое право и подсудность. |
|  3 | **Party Data Consistency Agent**             | Проверяет корректность и согласованность данных сторон: реквизиты, наименования, ИНН/ОГРН, адреса, полномочия подписантов, расхождения внутри документа.              |
|  4 | **Legal Mandatory Conditions Checker Agent** | Проверяет наличие и полноту обязательных условий по ГК РФ и другим применимым законам в зависимости от типа договора.                                                 |
|  5 | **Risk Detection & Severity Scoring Agent**  | Выявляет риск-конструкции в договоре и присваивает каждому риску уровень: высокий, средний или низкий.                                                                |
|  6 | **Recommendation Agent**                     | Формирует рекомендации и альтернативные формулировки по проблемным пунктам договора.                                                                                  |
|  7 | **Business Summary Agent**                   | Делает краткую выжимку договора простым языком для бизнес-пользователя: суть договора, ключевые условия, основные риски.                                              |
|  8  | **Detailed Report Agent**                    | Формирует детальный отчёт по результатам проверки: риск, уровень, место в документе, пояснение, основание, рекомендация.                                              |
|  9  | **Risk Delta Agent**                         | Сравнивает риск-профиль разных версий договора и показывает, какие риски появились, исчезли или изменили уровень.                          
                           
                                                                                 
Агентная цепочка
1. Contract Type Classifier Agent + Key Parameters Extractor Agent
        ↓
2. Party Data Consistency Agent
        ↓
3. Legal Mandatory Conditions Checker Agent + Risk Detection & Severity Scoring Agent
        ↓
4. Recommendation Agent
        ↓
5. Business Summary Agent + Detailed Report Agent
        ↓
6. Risk Delta Agent
