# Доменная декомпозиция

1) Document Proccessing 🔴 - stateless
   - загрузка и принятие файла
   - произведение OCR ( распознавание текста, извлечение текста ):
   - выделение разделов
   - построение semantic tree
   - сравнение версий
2) Legal Intelligence Core 🔴 ( Юридический мозг платформы. Множество нейронных сетей, которые производят анализ и формирование результатов ) - stateless
   - автоопределение типа договора и уровня уверенности результата определения
   - анализ и выявление рисков
   - создание пояснения по рискам
   - создание рекомендованных формулировок
   - расчет риск профиля
   - формирование сводной оценки договора
   - формирование краткого резюме
   - формирование детального отчета
3) Legal Knowledge Base 🟢 ( Источник правовых норм ГК РФ ) - statefull
   - загрузка юристом правовых норм ГК РФ
   - обучение нейронных сетей Legal Intelligence Core актуальными правовыми нормами
4) Organization Policy Management 🟡 ( Источник правовых норм и внутренних правил организации клиента. Конфигурация юридической политики клиента ) - statefull
   - загрузка шаблонов договоров организации, политик организации, чек-листов организации
   - настройка администратором строгости проверки, чек-листов обязательных условий, необходимых политик и т.д.
   - при работе Legal Intelligence CORE получает конкретную конфигурацию для анализа договора
5) Reporting Engine 🔴 ( Превращение юридических выводов системы в продуктовый результат для пользователя ) - stateless
   - формирование сводной оценки договора в пригодном для пользователя формате
   - формирование краткого резюме в пригодном для пользователя формате
   - формирование детального отчёта в пригодном для пользователя формате
   - формирование отчета по сравнению версий в пригодном для пользователя формате
   - экспорт PDF/DOCX
6) Document Management 🔴 ( Источник информации о договорах ) - statefull
   - версионность договора ( после каждого действия: загрузка, редактирование, применение рекомендаций, перепроверка сохраняется версия договора )
   - источник метаданных о договоре ( id договора, версия договора… )
   - хранение: OCR результат, semantic tree, сформированный детальный отчет, краткое резюме, сводная оценку, риск профиль
   - позволяет осуществлять сравнение версий
7) User & Organization Management 🟡 - statefull
   - авторизация, аутентификация
   - управление пользователями, ролями, правами доступа и привязкой к организациям-клиентам
8) Payment Processing 🟢 - statefull
   - произведение оплаты услуг сервиса клиентами
