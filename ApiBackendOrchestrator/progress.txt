- Broker.Prefetch >= 1
- Все duration > 0
- Upload.MaxSize > 0
- RateLimit: conditional (RPS > 0 только при Enabled=true)
- JWT: файл существует (os.Stat)
- LogLevel: enum (debug, info, warn, error)
- CircuitBreaker.FailureThreshold >= 1, MaxRequests >= 1
- Redis.PoolSize >= 1
